Map
Map API provide facility to convert SNOMED Concept Id's to other code system by specifying parameters.
SNOMED CT to ICD-10 Map -
Description - Converts given SNOMED concept id to corresponding ICD-10 code. Result includes mapped ICD-10 code and a list of advice with corresponding ICD-10 code.
Parameters -
  • id
  • gender
  • birthdate
  • diagnosisdate
  • secondary
